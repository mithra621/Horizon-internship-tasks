# 📈 Task 2 - Stock Portfolio Tracker

## Objective
Develop a Python program that calculates the total investment value of a stock portfolio using predefined stock prices.

## Features
- User enters stock names and quantities
- Uses a predefined dictionary of stock prices
- Calculates investment for each stock
- Displays the total investment value
- Saves the portfolio summary to a text file

## Technologies Used
- Python 3

## Concepts Used
- Dictionary
- User Input
- Loops
- Conditional Statements
- Basic Arithmetic
- File Handling

## How to Run

```bash
python stock_portfolio.py
```
