# 📧 Task 3 - Email Extractor

## Objective

Extract all valid email addresses from a text file and save them into a new file.

## Features

- Reads data from a text file
- Uses Regular Expressions (Regex)
- Finds valid email addresses
- Saves results to another text file
- Displays extracted emails

## Technologies Used

- Python 3

## Concepts Used

- Regular Expressions (`re`)
- File Handling
- Loops
- Strings

## How to Run

```bash
python email_extractor.py
```

## Output

The extracted email addresses are saved in:

```
extracted_emails.txt
```
