# 🤖 Task 4 - Basic Chatbot

## 📌 Objective

Build a simple rule-based chatbot in Python that responds to predefined user inputs using conditional statements.

## ✨ Features

- Greets the user with predefined responses
- Responds to common questions
- Runs continuously until the user types **"bye"**
- Handles unknown inputs with a default response
- Simple and beginner-friendly console application

## 🛠️ Technologies Used

- Python 3

## 📚 Concepts Used

- Functions
- While Loop
- If-Else Statements
- User Input and Output
- String Methods (`lower()`)

## 📂 Project Structure

## ▶️ How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python chatbot.py
```

## 💬 Sample Interaction

```
==================================================
🤖 Welcome to the Basic Chatbot!
Type 'bye' to exit the chat.
==================================================

You: hello
Bot: Hi! 👋

You: how are you
Bot: I'm fine, thanks! How about you?

You: what is your name
Bot: I'm a simple Python Chatbot.

You: bye
Bot: Goodbye! Have a great day! 👋
```

## 🎯 Learning Outcomes

- Understand how rule-based chatbots work
- Practice conditional statements (`if`, `elif`, `else`)
- Learn to use loops for continuous interaction
- Improve user input handling in Python
- Build a simple console-based application
